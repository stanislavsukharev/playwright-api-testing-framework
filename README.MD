# Playwright API Testing Framework

A framework template for API test automation in Playwright.

## Course Details

This framework is part of the course **"Playwright API Testing Mastery"** at **Bondar Academy**.  
The course includes lectures with a step-by-step implementation and a detailed explanation of how the framework is built and how it functions.

Learn Playwright API Testing from scratch.

Visit the course page here:  
**Playwright API Testing Mastery**

## Run Locally

### Install dependencies
```
npm install
```

Run sample test (smokeTest.spec.ts)
```
npx playwright test
```
